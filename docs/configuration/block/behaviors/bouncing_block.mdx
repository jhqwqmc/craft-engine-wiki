---
title: ⏏️ Bouncing Block
id: bouncing_block
---

**Bouncing Block** let you bounce off the block.

```yml
blocks:
  default:sofa:
    behavior:
      type: bouncing_block
      bounce-height: 0.66 # Bounce Height
      sync-player-position: true # Synchronize with player's own position. If the block you're using already has bounce functionality, set this to false
      fall-damage-multiplier: 0.5 # Fall damage multiplier
```
