---
title: ➡️ Face Attached Horizontal Directional Block
id: face_attached_horizontal_directional_block
---

| Property Name | Property Type        | Required |
|---------------|----------------------|----------|
| face          | anchor_type          | yes      |
| facing        | horizontal_direction | yes      |

```yaml
blocks:
  default:palm_button:
    behavior:
      type: face_attached_horizontal_directional_block
      blacklist: true
      attached-blocks:
        - custom:xxx
        - minecraft:stone
      attached-block-tags:
        - minecraft:dirt
        - minecraft:farmland
```
