---
title: 🌱 Budding Block
id: budding_block
---

**Budding Block** is used to generate or stimulate the growth of nearby blocks.

```yaml
blocks:
  default:infected_palm_log:
    behavior:
      type: budding_block
      growth_chance: 0.2
      blocks:
        - default:small_mushroom
        - default:medium_mushroom
        - default:large_mushroom
```
