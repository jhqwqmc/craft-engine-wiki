---
title: 2️⃣ Double High Block
id: double_high_block
---

**Double High Block** is a two-block-tall structure with lower and upper halves — like a tall flower. Placing one block automatically creates the upper half. Both halves must remain aligned: if the upper half loses its lower support, or the lower half loses the upper half, either piece breaks.
| Property Name | Property Type     | Required |
|---------------|-------------------|----------|
| half          | double_block_half | yes      |

## Example

```yaml
blocks:
  default:double_high_block:
    behavior:
      type: double_high_block
```
