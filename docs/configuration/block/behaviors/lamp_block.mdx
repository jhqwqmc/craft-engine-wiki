| Property Name | Property Type | Required |
|---------|---------|---------|
| lit    | boolean   | yes   |

**Lamp Block** allows the block to turn into `lit` state on redstone powered. 

```yaml
blocks:
  default:copper_coil:
    behavior:
      type: lamp_block
```
