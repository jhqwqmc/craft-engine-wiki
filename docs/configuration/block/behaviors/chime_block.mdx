---
title: 🔔 Chime Block
id: chime_block
---

**Chime Block** plays a sound when struck by a projectile (arrow, trident, fireball, etc.), like an amethyst block.

## Example

```yaml
blocks:
  default:palm_button:
    behavior:
      type: chime_block
      sounds:
        chime: minecraft:block.amethyst_block.chime   # sound played on projectile hit (optional)
```
