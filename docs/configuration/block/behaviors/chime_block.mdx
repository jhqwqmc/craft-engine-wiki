**Chime Block** emits a sound when struck by projectiles.

```yaml
blocks:
  default:palm_button:
    behavior:
      type: chime_block
      sounds:
        projectile-hit: minecraft:block.amethyst_block.chime
        # Or use `chime: minecraft:block.amethyst_block.chime`
```
