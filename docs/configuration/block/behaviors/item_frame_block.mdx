**Item Frame Block** is a type of block used for storing and displaying items. Used to replace the vanilla item frame, providing better performance. \
**Special Thanks: This feature was developed by [jhqwqmc](https://github.com/jhqwqmc/) with sponsorship from the [SIMMC server](https://simmc.cn/).**

```yml
blocks:
  default:item_frame_block:
    behavior:
      type: item_frame_block
      position: 0,0,0
      glow: false
      invisible: false
      render-map-item: true # Rendering map items is supported, but this requires additional performance overhead.
      sounds:
        put: entity.item_frame.add_item
        take: entity.item_frame.remove_item
        rotate: entity.item_frame.rotate_item
```
