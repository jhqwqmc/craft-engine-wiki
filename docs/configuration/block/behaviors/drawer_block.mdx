Drawer Block is a special type of storage container that can only hold one kind of item, but can store a large quantity of that same item.
Left-click to take out one item; sneak + left-click to take out a full stack.
Right-click to store a stack; continuously right-click to store all items from your inventory.

```yaml
blocks:
  default:drawer_block:
    behavior:
      type: drawer_block
      item_position: 0,0,-0.52
      text_position: 0,-0.4,-0.52
      item_scale: 0.6,0.6,0.6
      text_scale: 0.5,0.5,0.5
      max_stacks: 20
      sounds:
        put: minecraft:block.decorated_pot.insert
        take: minecraft:block.decorated_pot.insert_fail
      has_signal: true
      allow_input: true
      allow_output: true
      data_key: "craftengine:drawer"
      compatible_mode: false
```
