| Property Name | Property Type | Required |
|---------------|---------------|----------|
| high          | int           | yes      |

**Multi High Block** determines the block height based on the range difference of the `high` property.

```yaml
blocks:
  default:multi_high_block:
    states:
      properties:
        high:
          type: int
          default: 0
          range: 0~2
    behavior:
      type: multi_high_block
```
