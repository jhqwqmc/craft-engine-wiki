## Introduction

**Multi High Block Item** is a specialized placement method designed for multiple-block-tall blocks, fixing placement bugs that occur when using only the standard Block Item.

```yaml
items:
  default:palm_door:
    behavior:
      type: multi_high_block_item
      block: default:palm_door
```
