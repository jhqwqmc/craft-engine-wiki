---
title: 🔢 Multi High Block Item
id: multi_high_block_item
---

Like [block_item](block_item.mdx), but designed for blocks **three or more blocks tall**. Fixes placement offsets that occur with standard `block_item`.

```yaml
items:
  default:tall_plant:
    behavior:
      type: multi_high_block_item
      block: default:tall_plant
```
