## Introduction

**Ceiling Block Item** can only be placed on the bottom side of a block, all other behaviors are identical to the [Block Item](block_item.mdx).

```yaml
items:
  default:chandelier:
    behavior:
      type: ceiling_block_item
      block: default:chandelier
```
