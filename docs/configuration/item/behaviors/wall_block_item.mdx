## Introduction

**Wall Block Item** can only be placed on the side of a block, all other behaviors are identical to the [Block Item](block_item.mdx).

```yaml
items:
  default:palm_sapling:
    behavior:
      type: wall_block_item
      block: default:palm_sapling
```
