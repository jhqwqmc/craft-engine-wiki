## Overview

Renders multiple child models **layered on top of each other** in the same space. Order matters — later models render on top.

```yaml
items:
  demo:gem:
    material: diamond
    model:
      type: minecraft:composite
      models:
        - type: minecraft:model
          path: minecraft:item/custom/gem_base
        - type: minecraft:model
          path: minecraft:item/custom/gem_glow
```

## Fields

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `type` | string | Yes | — | `minecraft:composite` |
| `models` | list | Yes | — | List of child item models to render in order |
| `transformation` | object | No | — | Display transform overlay, composed with child transforms (26.1+) |
