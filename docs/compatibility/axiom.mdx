title: ⚛️ Axiom
id: axiom
---

First, ensure your server has [axiom-paper](https://modrinth.com/plugin/axiom-paper-plugin) installed.

Then download and install the [craft-engine-client-mod](https://github.com/Xiao-MoMi/craft-engine-client-mod/releases/latest).

Ensure the number of `server-side-blocks` configured on the client matches the number of `block.serverside-blocks` on the server.

Launch the client and enable the `Enable custom blocks in server` option from the mod menu.

For the server, simply enable the **`generate-mod-assets`** option in config.yml, and it will automatically generate client-side resources compatible with the mod.
