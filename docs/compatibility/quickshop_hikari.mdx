## Item Expression

Compatible with using [ItemExpression](https://quickshop-community.github.io/QuickShop-Hikari-Documents/docs/modules/item-expression) in QuickShop-Hikari to select the specific CraftEngine item.

```yaml
craftengine:default:palm_log
```
