:::tip
For advanced skript users, please use reflection to achieve more advanced functions. If you encounter problems when using the Skript features provided by CraftEngine, please [give us feedback](https://github.com/Xiao-MoMi/craft-engine/issues/new) in time. If you'd like to request a new Skript feature and you know Java, consider contributing your code via a pull request!
:::

:::warning
Please note that if you get an error when loading the server, it is probably because the skript script is loaded before the craftengine. This cannot be fixed at this time and can only be solved by reloading.
:::

## Conditions

<SkriptCard
  title="CraftEngine has been loaded"
  type="Condition"
  description="Checks CraftEngine has been loaded"
  syntax={[
    '(ce|craft[-]engine) (has been|is) load[ed]',
    '(ce|craft[-]engine) (has not been|is not) load[ed] [yet]',
    '(ce|craft[-]engine) (hasn\'t been|isn\'t) load[ed] [yet]'
  ]}
  examples={[
    `on script load:
      stop if craft-engine has not been loaded yet
      send "script init something!" to console`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Is Custom Item"
  type="Condition"
  description="Checks item is a custom item from CraftEngine"
  syntax={[
    '%itemstack/itemtype/slot% (is [a[n]]|are) (custom|ce|craft-engine) item[s]',
    '%itemstack/itemtype/slot% (isn\'t|is not|aren\'t|are not) [a[n]] (custom|ce|craft-engine) item[s]'
  ]}
  examples={[
    `on mine:
      if event-player's tool is a custom item:
        send "you are mine with a custom item" to event-player`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Is Custom Block"

  examples={[

<SkriptCard

## Expressions

<SkriptCard
  title="Custom Item"
  type="Expression"
  description="Get custom item from namespace id"
  syntax={[
    '[(the|a)] (custom|ce|craft-engine) item [with [namespace] id] %strings%',
  ]}
  examples={[
    `set {_item} to custom item with namespace id "default:topaz"`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Custom Item Id"
  type="Expression"
  description="Get a namespace id of custom item"
  syntax={[
    '(custom|ce|craft-engine) item [namespace] id of %itemstack/itemtype/slot%',
    '%itemstack/itemtype/slot%\'[s] (custom|ce|craft-engine) item [namespace] id'
  ]}
  examples={[
    `set {_itemId} to craft-engine item id of player's tool`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Custom Block Id"
  type="Expression"
  description="Get a namespace id of custom block"
  syntax={[
    '%blocks/blockdata/customblockstates%\'s (custom|ce|craft-engine) block [namespace] id',
    '(custom|ce|craft-engine) block [namespace] id of %blocks/blockdata/customblockstates%'
  ]}
  examples={[
    `on mine:
      if event-block's ce block id is "default:chinese_lantern":
        send "break a chinese_lantern block" to event-player`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Custom Block State"
  type="Expression"
  description="Get block state of custom block"
  syntax={[
    '%blocks/blockdata/customblockstates%\'s (custom|ce|craft-engine) block[ ]state',
    '(custom|ce|craft-engine) block[ ]state of %blocks/blockdata/customblockstates%'
  ]}
  examples={[
    `on mine:
      send event-block's ce block state to event-player`
  ]}
  defaultOpen={false}
/>

<SkriptCard
  title="Furniture Id"
  type="Expression"
  description="Get furniture id from entity"
  syntax={[
    '%entities%\'s [(custom|ce|craft-engine)] furniture [namespace] id',
  ]}
  examples={[
    `set {_furnitureId} to {_entity}'s craft-engine furniture id`
  ]}
  defaultOpen={false}
/>
