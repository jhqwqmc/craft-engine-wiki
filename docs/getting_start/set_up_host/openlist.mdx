:::warning
[OpenList](https://github.com/OpenListTeam/OpenList) is a community-driven fork of AList.
In June 2025, it was revealed that the AList project had been sold, and related download links were found to be compromised, sparking widespread concern and backlash within the community ([source](https://linux.do/t/topic/715019)).
Shortly thereafter, 123pan, 115, and Aliyundrive revoked their authorizations for AList, rendering some features unusable ([source](https://linux.do/t/topic/720642]).
Additionally, AList's original API services may have been included in the sale, posing a potential risk of information leakage.

Users still on AList are strongly advised to migrate to OpenList as soon as possible.
Migration guide: https://doc.oplist.org/guide/migrate
:::
