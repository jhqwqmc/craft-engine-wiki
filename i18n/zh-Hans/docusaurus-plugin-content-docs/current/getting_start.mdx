本教程带你从零开始，用 CraftEngine 给你的 Minecraft 服务器添加自定义物品、方块、家具和图片。每一章都能在游戏里看到亲手做出来的东西。

## 准备

开始前确保你有：

- 一个 Paper 系 Minecraft 服务端（1.20+），已装好 CraftEngine 插件
- 能编辑 YAML 的文本编辑器
- （可选）[BlockBench](https://www.blockbench.net/)，做模型用

如果插件还没装，先看[安装与托管](./getting_start/installation)。

## 关键命令

教程里反复用这两个：

| 情况 | 命令 |
|:---|:---|
| 改了 `.yml` 配置（物品、方块等） | `/ce reload config` |
| 改了合成配方 | `/ce reload recipe` |
| 改了模型/贴图文件 | `/ce reload all` |

> ⚠️ **最常见的坑**：改了贴图，跑了 `/ce reload config`，发现没变化——因为 `config` 不会重建资源包。贴图变更必须 `all`。

卡住了？多数错误 CraftEngine 都打印在控制台了，先看一眼。
