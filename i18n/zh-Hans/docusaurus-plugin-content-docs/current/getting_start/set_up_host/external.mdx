---
title: 外部托管
id: external
---

```yaml
resource-pack:
  delivery:
    hosting:
      - type: external
        url: ""
        uuid: "" # Optional
        sha1: "" # Optional
```

:::info

将资源包托管在外部服务器上。  
这可以避免资源包占用你自己服务器的带宽。

通常，在你将资源包上传到托管平台后，它会提供给你以下信息：

- URL（资源包下载链接）
- UUID（可选）
- SHA1（可选）

你只需要将这三个字段填写到配置中即可。

:::

:::caution

请注意，为了保持版本一致性，你需要定期更新资源包。这项维护操作需要你手动执行，不是自动完成的。

:::
