---
title: Alist
id: alist
---

```yaml
resource-pack:
  delivery:
    hosting:
      - type: alist
        api-url: "http://localhost:5244" # 填写你的站点地址
        username: "admin" # 填写该用户的用户名
        password: "your_password" # 填写该用户的密码
        upload-path: "/目录名称/server_resource_pack.zip" # 填写上传的到的绝对位置，以当前使用的用户可访问的根目录为准
        # 可选选项
        use-environment-variables: false # 是否使用环境变量提供机密信息，默认为 false
        disable-upload: false # 用于某些不支持上传的存储类型，默认为 false
        # file-password: "" # 访问文件的密码
        # otp-code: "" # 该用户的OTP验证码（不推荐开启因为这样你每隔 30 秒都要改一次）
        ## 可选代理配置
        # proxy:
        #   host: "127.0.0.1"
        #   port: 7890
```

<details>
  <summary>可用环境变量</summary>
  - CE_ALIST_USERNAME
  - CE_ALIST_PASSWORD
  - CE_ALIST_FILE_PASSWORD
</details>
