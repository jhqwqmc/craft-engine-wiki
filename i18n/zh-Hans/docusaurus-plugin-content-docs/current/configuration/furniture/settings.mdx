## 物品

确定此家具对应的物品。通常用于创造模式中键拾取（1.21.4+）。（默认：自动设为家具 ID）

```yaml
item: default:test_furniture
```

## 击打次数

确定玩家需要击打多少次才能破坏此家具。（默认：0，即第一次击打即刻破坏）

如果玩家停止击打超过 2 秒，或切换到击打另一件家具，击打计数会重置。

```yaml
hit_times: 3
```

## 音效

确定家具在各种情况下的音效（默认：无）

* break    当玩家破坏此家具时
* place    当玩家放置此家具时
* hit      当玩家击打此家具时

```yaml
sounds:
  break: minecraft:block.bamboo_wood.break
  place: minecraft:block.bamboo_wood.place
  hit: minecraft:block.bamboo_wood.hit
```

:::info
你可以像这样配置以精确控制音量和音调

```yaml
sounds:
  break:
    id: minecraft:block.deepslate.break
    pitch: 0.5
    volume: 0.25
  place: minecraft:block.deepslate.step
```
:::

## 冒险模式破坏

是否允许冒险模式的玩家破坏家具。（默认：false）

```yaml
adventure_mode_breaking: true
```

## 合适挖掘工具

确定高效破坏此家具所需的工具。（默认：无——任何工具或空手均可）

以 `#` 开头的值是物品**标签**（如 `#minecraft:axes` 匹配任何斧头）。不带 `#` 的值是具体的**物品 ID**（如 `minecraft:diamond_pickaxe`）。

```yaml
correct_tools:
  - "#minecraft:axes"
  - "minecraft:diamond_pickaxe"
```
