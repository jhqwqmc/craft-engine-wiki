## 物品

决定此家具对应的物品是什么。通常用于使用鼠标中键点击来选取物品（1.21.4+）。

```yaml
item: default:test_furniture # 默认为家具ID
```

## 击打次数

决定玩家需要击打多少次才能破坏此家具。

如果玩家停止击打超过 2 秒，或去击打其他家具，则击打计数会重置。

```yaml
hit_times: 3 # 默认为 0，即第一次击打即刻破坏
```

## 音效

决定家具在各种情况下的音效

* break    当玩家破坏此家具时
* place    当玩家放置此家具时
* hit      当玩家击打此家具时

```yaml
sounds: # 默认为 null
  break: minecraft:block.bamboo_wood.break
  place: minecraft:block.bamboo_wood.place
  hit: minecraft:block.bamboo_wood.hit
```

:::info
你可以像这样配置以精确控制音量和音调

```yaml
sounds:
  break:
    id: minecraft:block.deepslate.break
    pitch: 0.5
    volume: 0.25
  place: minecraft:block.deepslate.step
```
:::

## 冒险模式破坏

是否允许冒险模式的玩家破坏家具

```yaml
adventure_mode_breaking: true # 默认为 false
```

## 合适挖掘工具

从[代码](https://github.com/Xiao-MoMi/craft-engine/blob/23d2242ad1264cae5bea9c4033421236c6c11617/core/src/main/java/net/momirealms/craftengine/core/entity/furniture/setting/FurnitureSettings.java)来看未被使用（也就是无任何作用）

以 `#` 开头的值是物品**标签**（如 `#minecraft:axes` 匹配任何斧头）。不带 `#` 的值是具体的**物品 ID**（如 `minecraft:diamond_pickaxe`）。

```yaml
correct_tools:
  - "#minecraft:axes"
  - "minecraft:diamond_pickaxe"
```
