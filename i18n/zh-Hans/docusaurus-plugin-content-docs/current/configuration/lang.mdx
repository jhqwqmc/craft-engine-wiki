## 概览

此处 "language" 特指客户端语言设置，意味着文本内容会实时适配玩家当前客户端语言。

<UrlCard
  url="https://zh.minecraft.wiki/w/资源包#语言"
  title="语言"
/>

在 CraftEngine 中，你只需按照以下格式配置你的语言相关内容：

```yaml
lang:
  en_us:
    item.custom.palm_leaves: Palm Leaves
    item.custom.palm_log: Palm Log
```

:::tip

在缺少语言文件的情况下，Minecraft 会默认使用 `en_us`。因此，如果你正在创建**新**的翻译键，强烈建议配置 en_us。

如果你希望覆盖所有语言，请使用 "all" 作为语言。

```yaml
lang:
  all:
    container.inventory: ""
```

:::

:::info

如需渲染客户端翻译文本，请插入 [https://docs.papermc.io/adventure/minimessage/format/#translatable](https://docs.papermc.io/adventure/minimessage/format/#translatable)

示例：
```yml
items:
  custom:translate:
    material: paper
    data:
      item-name: "<lang:item.custom.palm_log>"
```
:::

## 翻译方块

除了自定义翻译键，CraftEngine 还提供了内置的方块名称快捷翻译，确保与以下组件兼容：

- CraftEngine 模组（客户端）
- 服务端方块翻译插件

配置方法非常简单。只需在常规方块命名空间ID前添加 `block_name:` 前缀，插件就会自动将其转换为对应的**真实方块命名空间ID**。

示例：`block_name:default:chinese_lantern` -> `block.craftengine.note_block_13`

```yaml
lang:
  en_us:
    block_name:default:chinese_lantern: Chinese Lantern
    block_name:default:netherite_anvil: Netherite Anvil
    block_name:default:topaz_ore: Topaz Ore
  zh_cn:
    block_name:default:chinese_lantern: 灯笼
    block_name:default:netherite_anvil: 下界合金砧
    block_name:default:topaz_ore: 黄玉矿石
```
