---
title: 🌊 液体碰撞方块物品
id: liquid_collision_block_item
---

与 [block_item](block_item.mdx) 类似，但允许放置在**液体表面**（水或熔岩）。使用 `offset_y` 调整相对于液面的放置高度。

```yaml
items:
  default:reed:
    material: paper
    behavior:
      type: liquid_collision_block_item
      offset_y: 1               # 相对于液面的高度偏移
      block: default:reed
```
