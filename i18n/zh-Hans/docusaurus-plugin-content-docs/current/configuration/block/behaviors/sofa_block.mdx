---
title: 🛋️ 沙发方块
id: sofa_block
---

**沙发方块**的行为类似可连接的座椅。其 `shape`（直形或内角）会根据后方是否有同类型沙发呈直角放置而自动检测。`shape` 在相邻方块变化时更新。

| 属性名称    | 属性类型             | 是否必需 |
|---------|------------------|------|
| shape   | sofa_shape        | 是    |
| facing  | horizontal_direction | 是    |
| waterlogged | boolean        | 否    |

## 示例

```yaml
blocks:
  default:sofa:
    behavior:
      type: sofa_block
```
