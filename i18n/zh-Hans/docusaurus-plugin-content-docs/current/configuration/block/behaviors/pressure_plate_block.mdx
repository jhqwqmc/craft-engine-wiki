---
title: ⬜️ 压力板方块
id: pressure_plate_block
---

**压力板方块**是一种由站在其上的实体触发的红石电源。当实体踩上时，输出满强度 **15** 的信号，在实体离开后持续保持 `pressed_time` 刻。

`sensitivity` 控制哪些实体可触发生效：
- **`all`（默认）：** 任意实体。
- **`mob`：** 仅限活体实体。

压力板需要下方有刚性或中心支撑方块。

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| powered | boolean | 是    |

## 示例

```yaml
blocks:
  default:palm_pressure_plate:
    behavior:
      type: pressure_plate_block
      sensitivity: all           # "all" 或 "mob"（默认 all）
      pressed_time: 20           # 实体离开后保持按下的时长，单位：刻（默认 20）
      sounds:
        on: minecraft:block.wooden_pressure_plate.click_on    # 按下时播放的声音（可选）
        off: minecraft:block.wooden_pressure_plate.click_off  # 释放时播放的声音（可选）
```

![](/img/pressure_plate_block.png)

:::info

信号恒为 **15**——仅开关信号，与原版测重压力板不同，不区分重量。

:::
