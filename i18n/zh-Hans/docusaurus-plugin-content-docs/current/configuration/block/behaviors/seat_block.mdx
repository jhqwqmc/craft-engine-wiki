<Highlight color="#3B6B60">**方块实体**</Highlight>​

| 属性名称   | 属性类型                 | 是否必需 |
|--------|----------------------|------|
| facing | horizontal_direction | 否    |

```yaml
blocks:
  default:sofa:
    behavior:
      type: seat_block
      seats:
        - 0,0,0 0   # x,y,z [角度]
```

```yaml
blocks:
  default:sofa:
    behavior:
      type: seat_block
      seats:
        - 0,0,0   
```

:::tip
x、y、z的数值会根据方块的朝向属性自动调整。

您会注意到在 `0,0,0` 后面还有一个额外的 `0`。如果填写此参数，表示将限制玩家在座位中的旋转角度。您可以尝试添加/移除该参数，并在椅子上环顾四周来感受差异。
:::
