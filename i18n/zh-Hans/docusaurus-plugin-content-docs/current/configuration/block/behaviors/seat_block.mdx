<Highlight color="#3B6B60">**方块实体**</Highlight>

**座椅方块**让玩家通过右键方块坐在上面。每个 `seats` 配置定义一个座位和一个可选[偏航角](https://zh.minecraft.wiki/w/%E6%9C%9D%E5%90%91#Y%E8%BD%B4%E6%97%8B%E8%BD%AC%E8%A7%92%EF%BC%88%E5%81%8F%E8%88%AA%E8%A7%92%EF%BC%89%E4%B8%8E%E6%96%B9%E4%BD%8D)旋转锁定角度。

若配置了 `facing` 属性会自动计算座椅的相对位置——位置随方块朝向旋转，因此同个坐标适用于所有方向。

| 属性名称   | 属性类型                 | 是否必需 |
|--------|----------------------|------|
| facing | horizontal_direction | 否    |

## 示例

```yaml
blocks:
  default:sofa:
    behavior:
      type: seat_block
      seats:
        - 0,0,0 0      # x,y,z [角度] —— 角度限制玩家身体旋转，省略则可自由旋转
```
