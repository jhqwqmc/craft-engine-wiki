<Highlight color="#3B6B60">**方块实体**</Highlight>

**座椅方块**让玩家通过右键点击坐在上面。每个 `seats` 条目定义一个坐位和一个可选 yaw 旋转锁定角度。

可选的 `facing` 属性会自动变换座椅位置——位置随方块朝向旋转，因此一组坐标适用于所有方向。

| 属性名称 | 属性类型             | 是否必需 |
|------|------------------|------|
| facing | horizontal_direction | 否    |

## 示例

```yaml
blocks:
  default:sofa:
    behavior:
      type: seat_block
      seats:
        - 0,0,0 0      # x,y,z [角度] —— 角度限制玩家视角旋转，省略则可自由环顾
```
