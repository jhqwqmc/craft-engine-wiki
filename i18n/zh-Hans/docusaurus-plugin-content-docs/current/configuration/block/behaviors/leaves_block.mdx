---
title: 🍁 树叶方块
id: leaves_block
---

**树叶方块**在与原木断开连接时会凋落。每片树叶持续计算其到最近原木的 `distance`：原木为 0，直接相邻的树叶为 1，以此类推。达到最大距离（`distance` 属性 range 中的最大值）的树叶会在随机刻中凋落并掉落物品。

将 `persistent` 设为 `true` 可完全阻止凋落。持久树叶不参与距离计算，无论距离多远都不会凋落。

| 属性名称    | 属性类型 | 是否必需 |
|---------|------|------|
| distance | int  | 是    |
| persistent | boolean | 是    |
| waterlogged | boolean | 否    |

## 示例

```yaml
blocks:
  default:palm_leaves:
    behavior:
      type: leaves_block
```

![](/img/leaves_block.png)

:::tip

对于较大的树木，可将 `distance` 属性的 range 设高一些——例如 `1~10` 的范围使树叶距原木最多 10 格才会凋落。原版树叶最大距离为 7。

:::
