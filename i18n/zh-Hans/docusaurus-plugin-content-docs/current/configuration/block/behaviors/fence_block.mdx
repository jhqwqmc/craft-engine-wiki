| 属性名称  | 属性类型    | 是否必需 |
|-------|---------|------|
| north | boolean | 是    |
| east  | boolean | 是    |
| south | boolean | 是    |
| west  | boolean | 是    |

```yaml
blocks:
  default:palm_fence:
    behavior:
      type: fence_block
      connectable-block-tag: minecraft:wooden_fences
      can-leash: true
```
