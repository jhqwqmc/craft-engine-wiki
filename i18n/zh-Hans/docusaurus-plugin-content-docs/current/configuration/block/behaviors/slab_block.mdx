---
title: ➖️ 台阶方块
id: slab_block
---

**台阶方块**行为与原版台阶一致。放置在方块下半部分或上半部分可创建下半台阶或上半台阶。将相同台阶放置在已有的单层台阶上会将两者合并为双层台阶（完整方块）。

双层台阶无法含水，液体也无法流入双层台阶。

| 属性名称    | 属性类型     | 是否必需 |
|---------|----------|------|
| type    | slab_type | 是    |
| waterlogged | boolean | 否    |

## 示例

```yaml
blocks:
  default:palm_slab:
     behavior:
       type: slab_block
```
