title: ➖️ 台阶方块
id: slab_block
---

| 属性名称        | 属性类型              | 是否必需 |
|-------------|-------------------|------|
| type        | slab_type         | 是    |
| half        | single_block_half | 是    |
| waterlogged | boolean           | 否    |

```yml
blocks:
  default:palm_slab:
     behavior:
       type: slab_block
```

![](/img/slab_block.png)

:::tip

最多可创建 5 种新台阶

:::
