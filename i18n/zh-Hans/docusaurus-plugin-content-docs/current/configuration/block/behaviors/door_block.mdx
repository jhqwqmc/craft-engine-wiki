---
title: 🚪 门方块
id: door_block
---

**门方块**是由下半部分与上半部分组成的两个方块高的结构。放置一个方块可自动创建两半。门可通过手、红石或风弹开启。

- **手：** 右键点击切换开关。
- **红石：** 开关与两半同步——上、下半同时切换。下半部分处理激活逻辑。
- **风弹：** 风弹击中下半部分可切换（仅在非红石充能时）。

`hinge` 侧在放置时根据相邻方块的碰撞形状自动判定，模拟原版门的铰链逻辑。

当其中一半被正确工具破坏或在创造模式中，另一半也会被移除，确保仅掉落一件物品。

| 属性名称    | 属性类型                 | 是否必需 |
|---------|----------------------|------|
| open    | boolean              | 是    |
| powered | boolean              | 是    |
| half    | double_block_half    | 是    |
| facing  | horizontal_direction | 是    |
| hinge   | hinge                | 是    |

## 示例

```yaml
blocks:
  default:palm_door:
    behavior:
      type: door_block
      can_open_with_hand: true        # 允许右键点击切换（默认 true）
      can_open_by_wind_charge: true   # 允许风弹切换（默认 true）
      sounds:
        open: block.wooden_door.open       # 打开时播放的声音（可选）
        close: block.wooden_door.close     # 关闭时播放的声音（可选）
```

![](/img/door_block.png)

:::warning

自定义门无法被村民或僵尸交互，即使添加相关方块标签。门的开关逻辑在 Minecraft 中属于硬编码，可能仅在后续版本中支持。

:::
