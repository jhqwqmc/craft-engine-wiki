---
title: 🚟 可悬挂方块
id: hangable_block
---

**可悬挂方块**既可放置于地面，也可悬挂在方块下方，就像原版灯笼。放置时会根据周围环境自动选择有效的放置方式。当对应方向的支撑方块被移除时，该方块会自动破坏。

| 属性名称        | 属性类型    | 是否必需 |
|-------------|---------|------|
| hanging     | boolean | 是    |
| waterlogged | boolean | 否    |

## 示例

```yaml
blocks:
  default:lantern:
    behavior:
      type: hangable_block
```
