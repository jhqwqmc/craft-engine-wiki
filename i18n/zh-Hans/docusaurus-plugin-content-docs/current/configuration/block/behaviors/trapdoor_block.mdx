---
title: 🪟 活板门方块
id: trapdoor_block
---

**活板门方块**是可开合的铰链面板。可通过手、红石或风弹切换。

- **手：** 右键点击开关。
- **红石：** 相邻信号激活时打开，信号切断时关闭。
- **风弹：** 风弹命中活板门会切换其状态。

放置时活板门自动检测红石与水——若被充能则打开，若在水中则含水。

| 属性名称    | 属性类型             | 是否必需 |
|---------|------------------|------|
| open    | boolean          | 是    |
| powered | boolean          | 是    |
| half    | single_block_half | 是    |
| facing  | horizontal_direction | 是    |
| waterlogged | boolean       | 否    |

## 示例

```yaml
blocks:
  default:palm_trapdoor:
    behavior:
      type: trapdoor_block
      can_open_with_hand: true        # 允许右键点击切换（默认 true）
      can_open_by_wind_charge: true   # 允许风弹切换（默认 true）
      sounds:
        open: block.wooden_trapdoor.open     # 打开时播放的声音（可选）
        close: block.wooden_trapdoor.close   # 关闭时播放的声音（可选）
```
