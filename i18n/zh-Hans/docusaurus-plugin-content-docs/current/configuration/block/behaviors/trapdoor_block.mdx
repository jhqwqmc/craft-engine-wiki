title: 🪟 活板门方块
id: trapdoor_block
---

| 属性名称        | 属性类型                 | 是否必需 |
|-------------|----------------------|------|
| open        | boolean              | 是    |
| powered     | boolean              | 是    |
| half        | single_block_half    | 是    |
| facing      | horizontal_direction | 是    |
| waterlogged | boolean              | 否    |

```yaml
blocks:
  default:palm_trapdoor:
    behavior:
      type: trapdoor_block
      can_open_with_hand: true
      can_open_by_wind_charge: true
      sounds:
        open: block.wooden_trapdoor.open
        close: block.wooden_trapdoor.close
```
