---
title: 🪻 灌木方块
id: bush_block
---

**灌木方块**是一种必须在特定支撑方块上生长的行为类型。若下方方块被破坏或处于无效位置，该方块将会破坏或掉落为物品。

```yaml
blocks:
  default:fairy_flower:
    behavior:
      type: bush_block
      stackable: false
      blacklist: false # 使用黑名单模式
      max-height: 0 # 大于 1 时启用
      delay: 0
      bottom-blocks:
        - custom:xxx
        - minecraft:stone
      bottom-block-tags:
        - minecraft:dirt
        - minecraft:farmland
```
