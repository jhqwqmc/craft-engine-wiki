---
title: 🍂 衰变方块
id: decay_block
---

**衰变方块**按 `age` 阶段逐步推进，最终转变为另一种方块——例如冰融化为水。

以两种模式之一工作：

- **固定间隔**（`delay` 设为正值或范围）：方块在计时器上运行。每段间隔后检查光照等级与 `chance`，若通过则增加 `age`，并重新安排下一次检查。`age` 达到 max 时，方块变为 `decay_into`。
- **随机刻**（`delay` 未设置或 ≤ 0）：方块改为监听随机刻。逻辑与上述相同。

若 `required_light` 设为大于 0，则仅在方块自身或周围光照达到阈值时才会衰变。

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| age  | int  | 是    |

## 示例

```yaml
blocks:
  default:ice:
    behavior:
      type: decay_block
      decay_into: water          # 完全衰变后变成的方块（默认 air）
      delay: 100~200             # 每次 age 推进的间隔（刻），省略则使用随机刻模式（可选）
      chance: 0.5                # 每次间隔推进的概率（默认 1）
      required_light: 0          # 衰变所需最低光照，0 = 始终衰变（默认 0）
```

:::tip

省略 `delay` 或将其设为非正值，行为会切换为**随机刻模式**——方块按随机刻而非固定计时器推进。

:::
