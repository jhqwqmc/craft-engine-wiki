---
title: 🔘 按钮方块
id: button_block
---

**按钮方块**是一种红石电源。被按下时开启并输出信号，片刻后自动弹起关闭。

可由以下方式触发：
- 玩家右键
- 射入其中的箭矢
- 风弹命中

输出始终为满强度 **15** 的信号，朝向按钮所附着的面。

## 属性

按钮本身只需要一个开/关状态。**定向红石输出依赖于 [`face_attached_horizontal_directional_block`](face_attached_horizontal_directional_block.mdx) 行为**——按钮会调用它来获知自身朝向。若方块上没有该行为，按钮仍能切换状态，但不会输出定向信号。

| 属性名称    | 属性类型             | 是否必需 | 来源行为                                   |
|---------|----------------------|------|--------------------------------------------|
| powered | boolean              | 是   | button_block                               |
| face    | anchor_type          | 是*  | face_attached_horizontal_directional_block |
| facing  | horizontal_direction | 是*  | face_attached_horizontal_directional_block |

\* 仅在需要定向红石输出时必需。参见[复合行为](behaviors.mdx#复合行为)。

:::note

`face` 与 `facing` 并非由按钮声明，而是来自与[面附着水平方向方块](face_attached_horizontal_directional_block.mdx)行为的组合。这是一种典型的复合行为依赖。

:::

## 示例

```yaml
blocks:
  default:palm_button:
    behaviors:
      - type: face_attached_horizontal_directional_block
        attached_block_tags:
          - minecraft:dirt
      - type: button_block
        ticks_to_stay_pressed: 30        # 按钮保持按下的时长，单位：刻（默认 30）
        can_be_activated_by_arrows: true # 射入按钮的箭矢是否让它保持按下（默认 true）
        sounds:
          on: minecraft:block.wooden_button.click_on   # 按下时播放的声音（可选）
          off: minecraft:block.wooden_button.click_off  # 释放时播放的声音（可选）
```
