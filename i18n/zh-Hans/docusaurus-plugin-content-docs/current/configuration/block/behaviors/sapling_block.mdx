---
title: 🌴 树苗方块
id: sapling_block
---

| 属性名称  | 属性类型 | 是否必需 |
|-------|------|------|
| stage | int  | 是    |

当指定 `configured_feature` 后，**树苗方块**可在随机刻事件中生长为指定的树木配置。

```yaml
blocks:
  default:palm_sapling:
    behavior:
      type: sapling_block
      feature: default:palm_tree
      bone_meal_success_chance: 0.45
      grow_speed: 0.7  # (0-1)
```

![](/img/sapling_block.png)

:::tip

树苗的生长阶段值越多，所需的生长时间越长。

:::
