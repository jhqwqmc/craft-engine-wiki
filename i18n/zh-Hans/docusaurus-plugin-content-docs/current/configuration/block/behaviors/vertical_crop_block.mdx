---
title: 🎍 垂直作物方块
id: vertical_crop_block
---

**垂直作物方块**会在随机刻中向上或向下生长，类似甘蔗或海带。

在垂直方向上，只有最顶端的茎段会增加 `age` 值。每次随机刻触发时，若生长方向的方块为空且总高度未达上限，则增加 `age` 值。当 `age` 值积累至最大值时，会在顶端延伸出一个新茎段，并将原顶端的 `age` 值重置为最小值。

## 属性

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| age  | int  | 是    |

## 示例

```yaml
blocks:
  default:flame_cane:
    behaviors:
      - type: vertical_crop_block
        max_height: 4            # 一株茎秆的最大段数（默认 3）
        grow_speed: 0.333        # 每随机刻增加的 age 值，小数部分为额外 +1 的概率（默认 1）
        direction: down          # 生长方向 up 或 down（默认 up）
      - type: hanging_block
        stackable: true
        delay: 1
```

:::info

`grow_speed` 决定茎段成熟的速度。整数部分每随机刻直接加给 `age`，小数部分作为概率提供额外 +1。例如 `0.333` 表示作物每刻有 33.3% 的概率使 `age` +1。

:::

:::warning

这是一个**随机刻**行为——只有方块收到随机刻时才会生长。

:::
