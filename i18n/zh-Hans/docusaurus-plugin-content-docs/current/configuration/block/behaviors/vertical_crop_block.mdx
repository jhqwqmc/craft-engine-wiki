---
title: 🎍 垂直作物方块
id: vertical_crop_block
---

**垂直作物方块**会在随机刻中向上或向下生长，类似甘蔗或海带。它累积内部的 `age` 值，达到最大值后在前方放置一个同类型的新方块。

根部在长出新的茎段后仍会继续生长——每一段独立生长，因此一株高大的茎秆可以在多个位置同时产生新的茎段。

仅当生长方向前方（`up` 时为上、`down` 时为下）为空气，且尖端下方已有的同方块茎段数少于 `max_height` 时，才会触发生长。

## 属性

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| age  | int  | 是    |

## 示例

```yaml
blocks:
  default:flame_cane:
    behaviors:
      - type: vertical_crop_block
        max_height: 4            # 一株茎秆的最大段数（默认 3）
        grow_speed: 0.333        # 每随机刻增加的 age 值，小数部分为额外 +1 的概率（默认 1）
        direction: down          # 生长方向："up" 或 "down"（默认 up）
      - type: hanging_block
        stackable: true
        delay: 1
```

![](/img/vertical_crop_block.png)

:::info

`grow_speed` 决定茎段成熟的速度。整数部分每随机刻直接加给 `age`；小数部分作为概率提供额外 +1。例如 `0.333` 表示作物每刻有 33.3% 的概率获得 1 点 age。

:::

:::warning

这是一个**随机刻**行为——只有方块收到随机刻时才会生长。

:::
