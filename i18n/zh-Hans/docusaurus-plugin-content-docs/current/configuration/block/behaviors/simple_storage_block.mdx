<Highlight color="#3B6B60">**方块实体**</Highlight>

**简单存储方块**是类似箱子的容器，可自定义行数。右键可打开其 GUI 存储物品。内置 0–15 比较器信号，反映容器填充程度。

可选的 `open` 属性用于追踪 GUI 是否打开——适合切换方块模型中的"打开"视觉状态。

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| open | boolean | 否    |

## 示例

```yaml
blocks:
  default:safe_block:
    behavior:
      type: simple_storage_block
      title: "保险柜"                   # GUI 标题，支持 MiniMessage 格式（默认 "<lang:container.chest>"）
      rows: 1                           # 行数，1~6（默认 1）
      has_signal: true                  # 根据填充比例输出比较器信号（默认 true）
      allow_input: true                 # 允许放入物品（默认 true）
      allow_output: true                # 允许取出物品（默认 true）
      data_key: "craftengine:simple_storage"   # 持久化数据的 NBT 键（默认 "craftengine:simple_storage"）
      sounds:
        open: minecraft:block.iron_trapdoor.open    # GUI 打开时播放的声音（可选）
        close: minecraft:block.iron_trapdoor.close   # GUI 关闭时播放的声音（可选）
```

![](/img/simple_storage_block.png)

:::info

该方块将物品栏暴露给漏斗等原版容器交互。`allow_input` 与 `allow_output` 同时控制手动操作与漏斗的存取。

:::
