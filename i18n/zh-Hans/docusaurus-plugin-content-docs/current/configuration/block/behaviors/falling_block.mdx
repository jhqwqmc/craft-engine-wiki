---
title: 🟨 可下落方块
id: falling_block
---

**可下落方块**在下方无方块时变为下落的实体，就像沙子或沙砾。放置后 2 刻内调度检查；若下方为空且高于最小世界高度，则生成使用该方块外观的下落实体。

当 `hurt_amount` 和 `max_hurt` 均为正数时，下落实体可伤害击中的实体。

## 示例

```yaml
blocks:
  default:falling_block_example:
    behavior:
      type: falling_block
      hurt_amount: -1             # 每次碰撞的伤害值，-1 = 禁用（默认 -1）
      max_hurt: -1                # 每次下落最多伤害的实体数，-1 = 禁用（默认 -1）
      sounds:
        land: minecraft:block.anvil.land       # 落地时播放的声音（可选）
        destroy: minecraft:block.anvil.destroy  # 破坏时播放的声音（可选）
```

![](/img/falling_block.png)

:::info

`hurt_amount` 与 `max_hurt` 必须一起使用——两者均需为正数（`> 0`），下落实体才会造成伤害。任一为 `-1` 或 `0` 则不造成伤害。

:::
