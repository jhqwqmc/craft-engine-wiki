---
title: 👾 扩散方块
id: spreading_block
---

**扩散方块**会在收到随机刻后尝试向周围扩散。每次尝试都会在附近随机选择一个位置，若该位置的方块类型与 `target_block` 相同，则将其替换为自身。

扩散范围为当前位置周围的随机区域（X/Z 方向 ±1 格，Y 方向 -3 至 +1 格）。

## 示例

```yaml
blocks:
  default:corruption_block:
    behavior:
      type: spreading_block
      target_block: minecraft:stone   # 可被转化的目标方块（必需）
```

:::warning

这是一个**随机刻**行为——只有方块收到随机刻时才会尝试扩散。

:::
