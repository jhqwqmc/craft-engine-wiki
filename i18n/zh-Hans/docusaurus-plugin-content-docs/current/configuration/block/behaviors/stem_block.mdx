---
title: 🍄 茎方块
id: stem_block
---

**茎方块**会随时间生长，就像原版西瓜或南瓜茎。每随机刻推进一个 `age` 阶段。完全成熟后选取随机水平方向，检查下方方块是否有效，然后在目标位置放置果实——同时将自身变为朝向该果实的 `attached_stem` 方块。

骨粉可跳过若干 age 阶段；若茎达到最大 age，会立即尝试生长果实。

仅在一定光照范围内生长——`light_requirement`（下限，默认 0）与 `max_light_requirement`（上限，默认 15）之间。

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| age  | int  | 是    |

## 示例

```yaml
blocks:
  default:hami_melon_stem:
    behavior:
      type: stem_block
      fruit: default:hami_melon                              # 茎生成的果实方块（必需）
      attached_stem: default:attached_hami_melon_stem        # 变为的附着茎方块（必需）
      light_requirement: 0                                   # 生长所需最低光照等级（默认 0）
      max_light_requirement: 15                            # 生长所需最高光照等级（默认 15）
      fruit_bottom_blocks:                                   # 果实可坐落的方块（白名单）
        - minecraft:farmland
        - minecraft:dirt
      fruit_bottom_block_tags: []                            # 果实可坐落的方块标签（白名单）
```

![](/img/stem_block.png)

:::info

`fruit_bottom_blocks` 与 `fruit_bottom_block_tags` 使用**白名单**——仅当目标位置下方的方块匹配时果实才会生长。空列表表示无方块满足条件。这与 `bush_block` 同时支持黑/白名单不同。

:::
