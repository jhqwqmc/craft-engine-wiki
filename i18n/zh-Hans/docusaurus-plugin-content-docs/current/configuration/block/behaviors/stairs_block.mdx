---
title: 🎢 楼梯方块
id: stairs_block
---

**楼梯方块**行为与原版楼梯一致。放置时根据点击位置自动设置 `facing` 和 `half`。其 `shape`（直梯、内角、外角）会在相邻方块变化时自动重新计算——楼梯通过检测相邻同半高楼梯来实现"连接"效果。

| 属性名称        | 属性类型                 | 是否必需 |
|-------------|----------------------|------|
| shape       | stairs_shape         | 是    |
| half        | single_block_half    | 是    |
| facing      | horizontal_direction | 是    |
| waterlogged | boolean              | 否    |

## 示例

```yaml
blocks:
  default:palm_stairs:
    behavior:
      type: stairs_block
```

![](/img/stairs_block.png)

:::warning

由于 Minecraft 的楼梯行为属于硬编码，自定义楼梯无法与原版楼梯自动无缝连接。

:::
