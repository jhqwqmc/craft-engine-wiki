**墙上火把粒子方块**按固定间隔生成粒子效果，类似原版的墙上的火把。`facing` 属性用于确定粒子相对于方块的位置，并随方块朝向自动旋转。

| 属性名称   | 属性类型      | 是否必需 |
|--------|-----------|------|
| facing | direction | 是    |

## 示例

```yaml
blocks:
  default:amethyst_wall_torch:
    behavior:
      type: wall_torch_particle_block
      tick_interval: 10 # 粒子生成间隔，单位：刻（默认为 10）
      particles:        # 具体选项见[✨ 粒子效果]页面
        - particle: smoke
          x: 0.5
          y: 0.7
          z: 0.5
          count: 1
          offset_x: 0
          offset_y: 0
          offset_z: 0
          speed: 0
```
