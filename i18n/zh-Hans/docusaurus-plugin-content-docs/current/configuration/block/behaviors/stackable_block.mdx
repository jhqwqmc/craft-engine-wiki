---
title: 🥪 可堆叠方块
id: stackable_block
---

**可堆叠方块**允许使用指定物品对已有方块进行堆叠，就像海泡菜或蜡烛。使用 `items` 列表中配置的物品对方块放置时，会递增指定 int 属性的值，直到达到该属性的最大值。不同的属性值可用于切换不同的视觉状态。

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| 自定义  | int  | 是    |

## 示例

```yaml
blocks:
  default:pebble:
    behavior:
      type: stackable_block
      property: pebble           # 用于记录堆叠数量的 int 属性名（默认为 "amount"）
      items:                     # 可用于增加堆叠的物品（必需）
        - default:pebble
```

:::info

`property` 指定用于记录堆叠数量的 int 属性。使用 `items` 中配置的物品对同类方块放置时，该属性会递增 1，直到达到其最大值。属性的取值范围决定了可用的堆叠等级数量。

:::
