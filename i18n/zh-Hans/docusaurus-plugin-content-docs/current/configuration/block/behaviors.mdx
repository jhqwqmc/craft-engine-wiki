## 简介

方块行为指的是方块独特的机制和物理特性。某些方块行为依赖于特定属性工作，例如：
- 作物方块需要 `age` 属性来追踪生长阶段
- 门需要 `hinge` 属性来确定左右部分

在后续页面中，你会看到类似这样的表格，其中列出了该方块行为所需的属性名称和类型。

| 属性名称        | 属性类型      | 是否必需 |
|-------------|-----------|------|
| facing      | direction | 是    |
| waterlogged | boolean   | 否    |

:::info

后续版本可能会支持自定义属性名，但当前实现使用硬编码名称以简化配置。
:::

## 复合行为

你可以组合多个行为来创建复合行为。以下是单行为和多行为方块的示例：

```yaml
blocks:
  default:fairy_flower:
    behavior:
      type: bush_block
      bottom-block-tags:
        - minecraft:dirt
        - minecraft:farmland
```
```yml
blocks:
  default:gunpowder_block:
    behaviors:
      - type: concrete_powder_block
        solid-block: default:solid_gunpowder_block
      - type: falling_block
```

:::warning

请注意：组合某些方块行为可能会导致意外冲突。如果遇到问题，请联系技术支持，我们将尽力解决这些冲突问题。

:::

:::caution

在组合多个方块行为时，必须同时合并它们所需的属性。

:::
